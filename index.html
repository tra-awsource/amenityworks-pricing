Use on your phone

  1. On your PC, run Start-Mobile-Server.ps1 (or double-click it).
  2. Phone on the same Wi‑Fi — open the URL it shows (e.g. http://192.168.x.x:8080).
  3. iPhone: Share → Add to Home Screen.
    Android: Menu → Install app / Add to Home screen.

Saved rates stay on this browser (phone and PC are separate unless you re-enter once on phone and Save defaults).

Server not required if you already have a public URL — bookmark that instead.

Offline-capable after first load on this network. Data stays in your browser (localStorage).
